# Privacy Policy for JobTracker Extension

**Last Updated: December 31, 2024**

## Overview
JobTracker is a Chrome extension that helps you track your LinkedIn job applications. We are committed to protecting your privacy.

## Data Collection
The extension collects and stores:
- Job application data (company name, job title, location, job URL, description)
- Your authentication credentials (email and JWT token)
- API configuration (backend URL, dashboard URL)

## Data Storage
- All data is stored locally in your browser using Chrome's storage API
- Authentication tokens are stored securely in Chrome's sync storage
- Pending job applications are stored in local storage

## Data Transmission
- Job application data is sent to YOUR configured backend API server
- We do NOT collect, store, or transmit your data to any third-party servers
- All communication happens between the extension and YOUR backend server

## Third-Party Services
- The extension only interacts with LinkedIn.com to detect job applications
- No data is shared with LinkedIn or any other third parties
- You control where your data is sent by configuring your own backend URL

## User Control
- You can delete all stored data by uninstalling the extension
- You can clear pending jobs at any time from the extension popup
- You control your backend server and database

## Permissions Used
- **storage**: To save your settings and pending job applications
- **notifications**: To notify you about tracked applications
- **host_permissions**: To access LinkedIn job pages and your configured backend API
